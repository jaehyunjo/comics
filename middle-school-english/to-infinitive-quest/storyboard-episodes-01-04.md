# 수한이의 30,000원 영어 퀘스트 — 1~4화 스토리보드

## 1화. 30,000원짜리 to 티켓!

- 목표: 명사·동사·동사원형을 소개하고 `to + 동사원형`의 첫 인상을 만든다.
- 핵심 예문: `I want eat pizza.` → `I want to eat pizza.`
- 컷 구성: 보상 미션 등장 → 수한이의 30,000원 상상 → 수인이의 틀린 문장 → 명사 설명 → 동사 설명 → 동사 충돌 개그 → `to` 티켓 등장 → `to eat` 변신 → 동사원형 설명 → `have/has` 예고 → 명사/형용사/부사 역할 카드 → 오늘의 핵심 정리 → QUEST CLEAR.
- 기초 박스: 명사=이름 붙일 수 있는 말, 동사=움직임/상태, 동사원형=기본 모양.

## 2화. 영어 문장의 기본 뼈대

- 목표: to부정사 전에 주어·동사·목적어와 영어 어순을 이해시킨다.
- 핵심 예문: `I eat pizza.`, `I like games.`, `I want to eat pizza.`
- 컷 구성: 문장 해골 개그 → 주어=누가 → 동사=한다/이다 → 목적어=무엇을 → `I + eat + pizza` 블록 조립 → 어순 오답 개그 → `want`와 `eat`의 충돌 복습 → `to eat`으로 해결 → QUEST CLEAR.
- 기초 박스: 주어, 동사, 목적어, 영어는 순서가 중요.

## 3화. have와 has의 비밀

- 목표: 주어에 따라 동사 모양이 달라질 수 있음을 맛보기로 배운다.
- 핵심 예문: `I have a book.`, `He has a book.`, `He wants to play.`
- 컷 구성: have/has 질문 → 주어 복습 → `I/you/we/they have` → `he/she/it has` → `He have` 오답 버저 → 3인칭 단수 맛보기 → `likes`의 s → 빈칸 퀴즈 → to부정사 문장과 연결 → QUEST CLEAR.
- 기초 박스: I/you/we/they have, he/she/it has, 한 명이면 s가 보일 때가 많다.

## 4화. 명사처럼 변신! ~하는 것

- 목표: to부정사의 명사적 용법을 주어·목적어·보어 자리로 소개한다.
- 핵심 예문: `To study English is fun.`, `I like to play games.`, `My dream is to be a pilot.`
- 컷 구성: to가 명사로 변신하는 장면 → `to study` = 공부하는 것 → 주어 자리 설명 → 긴 주어 예고 → 목적어 자리 `to play games` → 보어 자리 `to be a pilot` → 세 자리 의자 비유 → 미니 퀴즈 → QUEST CLEAR.
- 기초 박스: 명사적 용법=~하는 것, 주어=누가, 목적어=무엇을, 보어=설명 완성.
