## Character reference policy

When creating, editing, or prompting images for the middle-school math comics featuring 수한 and 수인, consult the dedicated reference before generating new pages or character art:

- `references/characters/suhan-suin.md`

Use the linked source character sheet as the primary visual reference:

- `middle-school-math/grade8-semester2-concept-manga/pages/00-suhan-suin-character-sheet.png`

Keep these traits consistent unless the user explicitly requests a redesign:

- 수한: 중2 형, 둥근 안경, 검은 후드/검은 캐주얼 복장, 차분하고 책임감 있는 노력형.
- 수인: 초6 동생, 남색 후드 집업, 순수하고 호기심 많은 성격, 형을 잘 따르는 현실 남매 케미.
- Shared style: 실사풍에 가까운 세밀한 일본 만화풍; avoid chibi/SD, overly childish proportions, or unexplained costume drift.

For image-generation prompts, include the reference traits in the prompt and keep official/source images as references rather than replacing them.
