# RFS 초보 파일럿 교실 — 첫 비행 편

모바일 비행 시뮬레이터 사용법을 게임 조작과 비행 원리 학습으로 함께 설명하는 한국어 교육 만화입니다.

> 주의: 공식 RFS 화면이나 로고를 복제하지 않고, 설명용 가상 비행 시뮬레이터 UI로 제작했습니다.

## 페이지 구성

1. `pages/01-why-airplanes-fly.png` — 비행기는 왜 뜰까: 추력·항력·양력·무게
2. `pages/02-reading-the-screen.png` — 화면 읽기: 속도·고도·자세·THR
3. `pages/03-throttle-and-speed.png` — THR: 속도를 만드는 첫 힘
4. `pages/04-flaps-and-lift.png` — FLAPS: 천천히 뜨는 날개
5. `pages/05-gear-and-drag.png` — GEAR: 하늘에서 바퀴는 저항
6. `pages/06-autopilot-goals.png` — 자동조종: 마법이 아니라 목표
7. `pages/07-landing-preparation.png` — 착륙 준비: 빨리 말고 정확하게
8. `pages/08-landing-complete.png` — 착륙 완료: 닿아도 끝이 아니다

## 제작 방향

- 실사풍에 가까운 세밀한 일본 만화풍 일러스트
- 중고등학생 미소년 주인공과 전문 교관 캐릭터
- 버튼 암기가 아니라 비행 원리 이해 중심
- 초보자가 게임을 하면서 추력, 항력, 양력, 무게, 플랩, 기어, 자동조종, 착륙 절차를 배워가도록 구성
