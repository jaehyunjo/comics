## 제작 메모

- 수한은 2012년생 중2, 키 약 177cm, 안경을 쓴 차분한 형입니다.
- 수인은 2017년생 동생, 축구광이고 장난꾸러기입니다.
- 수한의 자존심을 지키기 위해 성적·하위권 표현은 사용하지 않고, “개념을 정확히 이해하려는 학생”으로 표현합니다.
- 정우쌤은 논리와 풀이, 서윤쌤은 그림과 개념 설명을 담당합니다.
