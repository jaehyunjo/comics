# 장편 콘티: 수한이의 중2 1학기 부등식 천천히 정복

## 공통 연출

- 세로형 864x1821 학습만화.
- 한 페이지 5~7컷.
- 수한은 검은 후드, 둥근 안경, 차분한 표정. 수인은 남색 후드 집업, 질문하는 표정.
- 말풍선은 짧고 크게. 계산은 칠판, 노트, 수직선 카드에 또렷하게.
- 매 페이지 마지막 컷은 한 줄 정리.

## 01. 부등호가 무서운 이유

장면: 저녁 공부방. 수한이 문제집을 펴자 수인이가 옆에서 축구공을 안고 본다.

1. 수인: "형, 이 꺾인 표시는 왜 이렇게 많아?"
2. 수한: "부등호야. 사실은 '어느 쪽이 더 큰가'를 쓰는 표시야."
3. 책상 위 카드: `5 > 3`, `2 < 7`.
4. 수한: "입이 크게 벌어진 쪽이 큰 수를 본다고 생각해."
5. 수인: "그럼 5가 3보다 크니까 5쪽으로 입이 벌어지는구나."
6. 정리 박스: "부등호는 크기 비교를 짧게 쓰는 말."

## 02. 등식과 부등식은 무엇이 다를까

1. 저울 컷: 양쪽 사과가 같음. `3+2=5`.
2. 수한: "등식은 양쪽 값이 정확히 같다는 말."
3. 저울 컷: 한쪽이 무거움. `x+2>5`.
4. 수한: "부등식은 같다는 뜻이 아니라, 어느 범위가 가능하다는 뜻이야."
5. 수인: "답이 하나가 아닐 수도 있어?"
6. 수한: "맞아. 부등식은 답 후보들의 모임을 찾는 문제야."

## 03. 부등호 네 가지 읽기

1. 칠판 네 칸: `<`, `>`, `<=`, `>=`.
2. 수한: "`<`는 작다, `>`는 크다."
3. 수한: "`<=`는 작거나 같다. 경계가 포함돼."
4. 수인: "`>=`는 크거나 같다. 이것도 경계 포함!"
5. 비교 카드: `x<4`, `x<=4`.
6. 정리: "`같다`가 붙으면 경계 숫자도 답."

## 04. 수직선은 범위를 보여 주는 지도

1. 책상 위 긴 종이 수직선.
2. 수한: "`x<4`는 4보다 왼쪽 전체."
3. 열린 점으로 4 표시.
4. 수인: "4는 안 되니까 점을 비워 두는구나."
5. `x<=4`는 닫힌 점.
6. 정리: "열린 점: 경계 제외. 닫힌 점: 경계 포함."

## 05. x는 한 숫자가 아니라 후보들

1. 문제: `x>3`.
2. 수인: "답은 4야?"
3. 수한: "4도 맞지만 5도, 10도, 3.5도 맞아."
4. 수직선 위 여러 점이 빛남.
5. 수한: "부등식의 해는 조건을 만족하는 모든 수야."
6. 정리: "해 하나가 아니라 해의 범위를 찾는다."

## 06. 대입해서 맞는지 확인하기

1. 문제: `x+2<7`.
2. 수인: "x=3이면?"
3. 노트: `3+2=5`, `5<7`, 맞음.
4. 수인: "x=6이면?"
5. 노트: `6+2=8`, `8<7`, 틀림.
6. 수한: "헷갈리면 먼저 넣어 봐. 부등식도 검산이 돼."

## 07. 양쪽에 같은 수를 더해도 되는 이유

1. 저울 그림: `3<5`.
2. 양쪽에 블록 2개씩 추가: `3+2<5+2`.
3. 수한: "둘 다 똑같이 커지면 순서는 그대로야."
4. 계산: `x-2<6`, 양쪽에 2 더하기.
5. `x<8`.
6. 정리: "양쪽에 같은 수를 더해도 부등호 방향은 그대로."

## 08. 양쪽에서 같은 수를 빼도 되는 이유

1. 줄 서 있는 키 비교: 170cm < 180cm.
2. 둘 다 10cm 낮은 받침대를 내려옴: 160cm < 170cm.
3. 수한: "같이 줄어들면 순서는 안 바뀌어."
4. 문제: `x+5>=12`.
5. 양쪽에서 5 빼기: `x>=7`.
6. 정리: "같은 수를 빼도 방향은 그대로."

## 09. 곱하기와 나누기도 균형을 지킨다

1. `2<4` 카드.
2. 양쪽에 3배: `6<12`.
3. 수한: "양수를 곱하면 둘 다 같은 비율로 커져."
4. 문제: `x/3<5`.
5. 양쪽에 3 곱하기: `x<15`.
6. 정리: "양수로 곱하거나 나누면 방향 그대로."

## 10. 음수를 곱하면 왜 방향이 바뀔까

1. 수직선 확대: `2<4`.
2. 둘 다 -1을 곱해 `-2`와 `-4`로 이동.
3. 수인: "어? -2가 -4보다 오른쪽이네!"
4. 수한: "음수를 곱하면 수직선에서 좌우가 뒤집혀."
5. `2<4` -> `-2>-4`.
6. 정리: "음수 곱셈/나눗셈은 부등호 방향을 바꾼다."

## 11. 실수 방지: 마지막 줄을 꼭 읽자

1. 수한이 빨간 펜으로 `-2x<8` 표시.
2. 양쪽을 -2로 나눔.
3. 잘못된 풀이: `x< -4`.
4. 수한: "여기서 방향을 바꿔야 해."
5. 바른 풀이: `x> -4`.
6. 정리: "나눈 수가 음수인지 마지막에 확인."

## 12. 일차부등식의 모양

1. 칠판: `2x+3<11`.
2. 수한: "x가 1제곱으로만 나오면 일차부등식이야."
3. `x^2`는 아직 아님 표시.
4. 수인: "목표는 x 혼자 남기기?"
5. 수한: "정확해. x의 범위를 찾는 거야."
6. 정리: "일차부등식 풀이 목표: x만 남기기."

## 13. x만 남기는 목표

1. 수한이 상자 속 x 그림을 보여 줌.
2. 문제: `x+4<10`.
3. 수한: "x 옆에 붙은 4를 없애려면 양쪽에서 4를 빼."
4. `x<6`.
5. 수인: "x를 혼자 두려고 반대 행동을 하는구나."
6. 정리: "더하기는 빼기로, 곱하기는 나누기로 풀어낸다."

## 14. 2x+3<11 풀기

1. 문제 크게: `2x+3<11`.
2. 수한: "먼저 +3을 없애자."
3. `2x<8`.
4. 수한: "2x는 x가 2묶음이라는 뜻. 양쪽을 2로 나눠."
5. `x<4`.
6. 검산: x=3은 맞고 x=5는 틀림.

## 15. -3x+5<=14 풀기

1. 문제: `-3x+5<=14`.
2. `+5` 없애기: `-3x<=9`.
3. 수인: "이제 -3으로 나누면..."
4. 수한: "음수로 나누니까 방향 바꿔."
5. `x>=-3`.
6. 정리: "계수가 음수면 마지막 방향 전환."

## 16. 분수가 나오면 어떻게 할까

1. 문제: `x/4 + 2 > 5`.
2. `x/4 > 3`.
3. 수한: "4로 나눈 걸 없애려면 양쪽에 4를 곱해."
4. `x>12`.
5. 수인: "4는 양수니까 방향 그대로!"
6. 정리: "분모가 양수면 곱해도 방향 그대로."

## 17. 소수가 나오면 어떻게 할까

1. 문제: `0.2x + 1 <= 3`.
2. 수인: "소수는 보기만 해도 어려워."
3. 수한: "양쪽을 10배 해서 정수로 바꿔도 돼."
4. `2x+10<=30`.
5. `2x<=20`, `x<=10`.
6. 정리: "소수는 10배, 100배로 깨끗하게."

## 18. 괄호가 있으면 먼저 펼칠까

1. 문제: `3(x-2)<12`.
2. 방법 A: 먼저 3으로 나누기 `x-2<4`.
3. 방법 B: 괄호 풀기 `3x-6<12`.
4. 수한: "둘 다 되지만 쉬운 길을 고르면 돼."
5. `x<6`.
6. 정리: "괄호는 풀어도 되고, 먼저 나눠도 된다. 실수 적은 쪽 선택."

## 19. 양쪽에 x가 있을 때

1. 문제: `5x-1 < 2x+8`.
2. 수한: "x는 한쪽으로, 숫자는 반대쪽으로 모아."
3. 양쪽에서 2x 빼기: `3x-1<8`.
4. 양쪽에 1 더하기: `3x<9`.
5. `x<3`.
6. 정리: "문자끼리 먼저 모으면 길이 보인다."

## 20. 풀이 순서 체크리스트

1. 수한 노트 체크리스트.
2. 1 괄호 정리.
3. 2 분수/소수 정리.
4. 3 문자 한쪽, 숫자 한쪽.
5. 4 x의 계수로 나누기.
6. 5 음수로 나눴는지 확인.

## 21. 수직선에 답을 그리기

1. 답: `x>=-2`.
2. -2에 닫힌 점, 오른쪽 화살표.
3. 답: `x<5`.
4. 5에 열린 점, 왼쪽 화살표.
5. 수인: "그림으로 보니 답이 어디까지인지 보여."
6. 정리: "답을 식과 수직선 둘 다로 확인."

## 22. 말로 된 문제 첫걸음

1. 문장제: "어떤 수에 3을 더하면 10보다 작다."
2. 수인: "어떤 수가 x?"
3. 수한: "좋아. '3을 더하면'은 x+3."
4. "10보다 작다"는 `<10`.
5. `x+3<10`, `x<7`.
6. 정리: "모르는 양을 x로 두고 문장을 순서대로 번역."

## 23. 이상과 초과는 다르다

1. 풋살장 키 제한 안내판: "140cm 이상".
2. 수한: "이상은 그 숫자도 포함."
3. `x>=140`.
4. "140cm 초과" 안내판.
5. 수인: "초과는 140은 안 되고 넘겨야 해."
6. 정리: "이상: >=, 초과: >."

## 24. 이하와 미만은 다르다

1. 도서관 대출 제한: "5권 이하".
2. `x<=5`.
3. 간식 봉지 문구: "500원 미만".
4. `x<500`.
5. 수한: "이하에는 같다가 붙고, 미만에는 같다가 없어."
6. 정리: "이하: <=, 미만: <."

## 25. 용돈 문제

1. 수인: "5000원으로 연필과 지우개를 사고 싶어."
2. 연필 1200원, 지우개 x원.
3. 수한: "총액이 5000원 이하."
4. `1200+x<=5000`.
5. `x<=3800`.
6. 정리: "예산 문제는 총액 <= 가진 돈."

## 26. 과자 개수 문제

1. 과자 한 봉지 700원, 가진 돈 5000원.
2. 수인: "몇 봉지까지 살 수 있어?"
3. `700x<=5000`.
4. `x<=50/7`, 약 7.14.
5. 수한: "봉지는 자연수니까 최대 7봉지."
6. 정리: "개수 문제는 정수 조건을 마지막에 확인."

## 27. 버스 요금 문제

1. 교통카드 9000원, 한 번 1450원.
2. `1450x<=9000`.
3. `x<=9000/1450`.
4. 계산기 없이 6번은 8700원, 7번은 10150원.
5. 수인: "최대 6번!"
6. 정리: "나누기 어려우면 직접 가까운 배수를 비교."

## 28. 축구 연습 시간 문제

1. 수인이 풋살장에서 공을 차고 있음.
2. 숙제 40분, 샤워 20분, 남은 시간 120분.
3. 축구 시간 x분. `x+40+20<=120`.
4. `x<=60`.
5. 수인: "한 시간까지 가능!"
6. 정리: "전체 시간 안에 들어가야 하면 합 <= 전체."

## 29. 평균 점수 문제

1. 세 번 시험: 70, 80, x.
2. 평균이 80점 이상.
3. `(70+80+x)/3 >= 80`.
4. `150+x>=240`.
5. `x>=90`.
6. 정리: "평균 문제는 합계로 바꿔라."

## 30. 할인 문제

1. 12000원 문제집, 쿠폰 x원 할인, 예산 9000원.
2. `12000-x<=9000`.
3. `-x<=-3000`.
4. 음수 나누기: `x>=3000`.
5. 수인: "할인은 적어도 3000원 필요!"
6. 정리: "빼는 x는 음수 계수 주의."

## 31. 속력·거리·시간 문제

1. 자전거로 12km, 시속 x km.
2. 1시간 30분 안에 도착.
3. 시간 = 거리/속력, `12/x <= 1.5`.
4. 중2 기본 범위에서는 표로 접근: x=6이면 2시간, x=8이면 1.5시간.
5. `x>=8`.
6. 정리: "비례식이 낯설면 표로 먼저 감을 잡아라."

## 32. 둘 중 어느 쪽이 이득일까

1. A 요금제: 기본 2000원 + 1회 500원.
2. B 요금제: 기본 5000원 + 1회 200원.
3. x회 이용할 때 B가 더 싸려면 `5000+200x < 2000+500x`.
4. `3000<300x`.
5. `x>10`.
6. 정리: "비교 문제는 한쪽 비용 < 다른쪽 비용."

## 33. 두 조건을 동시에

1. 문제: x는 3 이상이고 8 미만.
2. `x>=3`, `x<8`.
3. 수직선에서 겹치는 부분 색칠.
4. `3<=x<8`.
5. 수인: "둘 다 만족하는 구간만 남기네."
6. 정리: "동시에 만족은 겹치는 범위."

## 34. 정수해를 찾는 법

1. 답: `-2<x<=3`.
2. 수직선에 -2 열린 점, 3 닫힌 점.
3. 정수만 표시: -1, 0, 1, 2, 3.
4. 수인: "경계가 열렸는지 닫혔는지 먼저 봐야 해."
5. 정수해 5개.
6. 정리: "정수해는 수직선 위 정수 점을 세기."

## 35. 자연수해를 찾는 법

1. 답: `x<5`.
2. 수인: "그럼 4.5도?"
3. 수한: "자연수라면 1,2,3,4만."
4. 0은 학교 교과서 조건에 따라 자연수에서 제외한다고 표시.
5. 자연수해 4개.
6. 정리: "자연수 조건은 마지막에 따로 걸러라."

## 36. 실전 문제 읽는 순서

1. 긴 문장제 등장.
2. 수한: "첫째, 무엇을 x로 둘지 찾기."
3. "둘째, 제한 조건에 밑줄."
4. "셋째, 이상/이하/초과/미만 표시."
5. "넷째, 단위 확인."
6. 정리: "읽는 순서가 풀이의 절반."

## 37. 자주 틀리는 말 바꾸기

1. 단어 카드: 적어도, 많아야, 넘게, 이내.
2. 적어도 5개 -> `x>=5`.
3. 많아야 5개 -> `x<=5`.
4. 5개 넘게 -> `x>5`.
5. 5개 이내 -> `x<=5`.
6. 정리: "말을 부등호로 번역하는 연습이 중요."

## 38. 오답노트 1: 부등호 반대

1. 잘못된 풀이: `-4x>12` -> `x> -3`.
2. 수인이 빨간 펜으로 갸웃.
3. 수한: "음수 -4로 나눴으니 방향을 바꿔야 해."
4. 정답: `x< -3`.
5. x=-4 대입 검산.
6. 정리: "방향이 의심되면 아무 수나 넣어 검산."

## 39. 오답노트 2: 경계점 실수

1. 답 `x<=2`인데 열린 점으로 그림.
2. 수인: "2도 답인데 점이 비어 있어."
3. 닫힌 점으로 수정.
4. `x<2`와 비교.
5. 수한: "같다가 있으면 닫힌 점."
6. 정리: "부등호 밑줄은 닫힌 점 신호."

## 40. 오답노트 3: 질문을 안 읽음

1. 계산 결과: `x<=6`.
2. 문제 질문: "살 수 있는 최대 개수는?"
3. 수인이 답을 `x<=6`이라고 적음.
4. 수한: "질문은 최대 개수니까 6개라고 써야 해."
5. 답칸: `최대 6개`.
6. 정리: "마지막 문장을 보고 답의 모양을 맞춰라."

## 41. 중간 보스 문제

1. 문제: "입장료 3000원, 놀이기구 1회 1200원. 10000원 이하로 쓰려면 몇 회까지?"
2. x회 설정.
3. `3000+1200x<=10000`.
4. `1200x<=7000`.
5. 5회 6000원, 6회 7200원이라 최대 5회.
6. 정리: "나눠서 소수가 나오면 실제 개수 조건으로 마무리."

## 42. 수한이의 풀이 말하기

1. 수한이 칠판 앞에서 풀이 순서를 소리 내어 말함.
2. "모르는 놀이기구 횟수를 x로 둔다."
3. "총금액은 입장료 더하기 회당요금 곱하기 횟수."
4. "10000원 이하니까 <=."
5. "계산 후 x가 자연수인지 확인."
6. 정리: "풀이를 말로 설명할 수 있으면 진짜 이해한 것."

## 43. 수인이가 직접 풀어 보기

1. 문제: "한 봉지 800원 과자를 5000원 이하로 사려면?"
2. 수인: `800x<=5000`.
3. `x<=6.25`.
4. "과자는 자연수니까 최대 6봉지."
5. 수한이 고개를 끄덕임.
6. 정리: "식, 계산, 조건, 답 문장."

## 44. 수인이가 막힌 문제

1. 문제: "현재 55점, 다음 시험 x점. 두 시험 평균이 70점 이상."
2. 수인: "평균에서 막혔어."
3. 수한: "평균은 합계를 개수로 나눈 것."
4. `(55+x)/2>=70`.
5. `55+x>=140`, `x>=85`.
6. 정리: "평균은 무조건 합계부터."

## 45. 형제 합동 풀이

1. 문제: "쿠폰 2000원을 쓰고도 15000원 이하. 한 권 3500원 책 x권."
2. 총액: `3500x-2000<=15000`.
3. `3500x<=17000`.
4. x<=약 4.85.
5. 최대 4권.
6. 정리: "할인/쿠폰은 총액에서 빼기."

## 46. 시험 직전 5분 정리

1. 수한 노트 한 페이지.
2. 부등호 네 가지.
3. 음수 곱셈/나눗셈 방향 전환.
4. 수직선 열린 점/닫힌 점.
5. 문장제 4단계: x 정하기, 식 세우기, 풀기, 조건 확인.
6. 정리: "부등식은 범위를 찾는 문제."

## 47. 최종 보스

1. 장문 문제: "수인이가 풋살장까지 버스와 간식을 포함해 12000원 이하로 쓰려 한다. 왕복 버스비 2900원, 음료 1500원, 풋살장 이용권은 1시간 2000원. 몇 시간까지?"
2. x시간 설정.
3. `2900+1500+2000x<=12000`.
4. `4400+2000x<=12000`.
5. `2000x<=7600`, x<=3.8.
6. 답: "최대 3시간."

## 48. 에필로그

1. 수인이 풋살장에서 슛을 넣음.
2. 수한이 노트에 `x<=3`을 적으며 웃음.
3. 수인: "부등식은 답 하나가 아니라 가능한 범위를 찾는 거네."
4. 수한: "그리고 문제는 천천히 번역하면 풀려."
5. 둘이 집으로 걸어가며 다음 단원 예고.
6. 마지막 정리: "문장 -> 부등식 -> 풀이 -> 범위 -> 답 문장."
