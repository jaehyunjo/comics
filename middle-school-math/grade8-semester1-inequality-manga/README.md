# 수한이의 중2 1학기 부등식 천천히 정복

중학교 2학년 1학기 `부등식` 단원을 아주 기초부터 따라가도록 설계한 장편 학습만화입니다.

기존 수한·수인 만화와 같은 세로형 풀컬러 학습만화 형식을 기준으로 합니다.

## 캐릭터 기준

- 수한: 중2 형, 약 177cm, 검은 짧은 머리, 둥근 안경, 검은 후드와 검은 캐주얼 복장, 차분하고 책임감 있는 노력형 주인공.
- 수인: 수한의 친동생, 약 140cm, 남색 후드 집업, 호기심 많고 질문을 잘함. 축구를 잘하고 수원 삼성 팬.
- 형제 관계: 수한은 정리하고 설명하는 형, 수인은 질문하고 발견하는 동생.
- 스타일: 실사풍에 가까운 세밀한 일본 만화풍, 따뜻한 학습만화 톤, 치비/SD 금지.

자세한 기준은 `../../references/characters/suhan-suin.md`와 `../grade8-semester2-concept-manga/pages/00-suhan-suin-character-sheet.png`를 따른다.
